# High
De aanvaller heeft ons opnieuw door. We kunnen nog meer tekens niet gebruiken:

![tip](tip_high.png)

Start `DVWA` en ga naar de challenge `Command Injection`. Zorg dat je `DVWA Security` op `high` hebt staan.
Ben je vergeten hoe dit moest? Ga dan naar de [cheatsheet](../../docs/cheatsheet).

Bij één van de gefilterde tekens heeft de programmeur een typefout gemaakt. Zie jij welke dit is en hoe je alsnog `command injection` kunt uitvoeren?

Het resultaat van een succesvolle poging zie je hieronder:

![succes](high_succes.png)

## Oefen met de terminal

In deze terminal worden bijna alle operators gefilterd. Maar let goed op: de programmeur filtert `"| "` (pipe + spatie), niet `"|"` (pipe zonder spatie). Kun je dit misbruiken?

<LinuxTerminal
  title="Command Injection — High (streng gefilterd)"
  filter={(input) => {
    let filtered = input;
    filtered = filtered.replace(/&&/g, '');
    filtered = filtered.replace(/;/g, '');
    filtered = filtered.replace(/\|\| /g, '');
    filtered = filtered.replace(/\| /g, '');
    return filtered;
  }}
/>
